# Level 1 - Java Development Internship

This folder contains the **Level 1 tasks** completed as part of my **Java Development Internship**. These projects helped me build a strong foundation in Java programming, problem-solving, and object-oriented programming concepts.

---

## 📌 Projects

### 🧮 Task 1: Basic Calculator
A console-based Java application that performs basic arithmetic operations.

**Features:**
- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Menu-driven interface
- Input validation

---

### 🎯 Task 2: Number Guessing Game
A Java game where the user guesses a randomly generated number within a limited number of attempts.

**Features:**
- Random number generation
- User input validation
- "Too High" and "Too Low" hints
- Limited attempts
- Win/Lose messages

---

### 🔁 Task 3: Recursion Programs
A collection of Java programs demonstrating recursion and its applications.

**Topics Covered:**
- Factorial using Recursion
- Fibonacci Series
- Sum of Natural Numbers
- Reverse a String
- Power Calculation
- Other recursion-based problems
